E-Commerce Sales Dashboard – Power BI 

This project focuses on building an interactive sales analytics dashboard using Power BI to analyze key business metrics from an e-commerce dataset. The goal was to transform raw sales data into meaningful insights that help understand sales performance, customer purchasing behavior, and product profitability. 

Project Objectives:-
-> Clean and prepare raw e-commerce data for analysis  
-> Identify key sales and profit trends  
-> Analyze customer purchasing behavior  
-> Provide business insights through interactive visualizations  

Data Preparation:-
The raw dataset required several preprocessing steps before analysis: 
-> Removed duplicate records and corrected inconsistent data entries 
-> Handled missing values to improve dataset reliability 
-> Standardized categorical fields such as State, Category, Payment Mode, and Sub-Category  
-> Performed data transformation and structuring using Power Query 

Dashboard Features:- 
The Power BI dashboard provides multiple analytical views including:  
Key Performance Indicators (KPIs):- 
-> Total Sales: ₹438K  
-> Total Profit: ₹37K  
-> Total Quantity Sold: 5615  
-> Average Order Value (AOV): ₹121K  

Visual Insights: 
-> Sales by State to identify top-performing regions 
-> Profit by Month to track monthly profit trends  
-> Sales by Customer to identify high-value customers 
-> Quantity by Payment Mode to understand payment preferences 
-> Sales by Product Category
-> Profit by Sub-Category to identify the most profitable products 

Interactive Features:
-> State Filter for regional analysis 
-> Quarter Slicer to explore quarterly performance  

Key Insights:- 
-> Clothing contributes the largest share of total quantity sold.  
-> Cash on Delivery (COD) is the most preferred payment mode.  
-> Certain sub-categories such as printers and bookcases generate higher profits. 
-> Sales performance varies significantly across states and months.  
-> Tools Used  Power BI  Power Query  Data Cleaning  Data Visualization  Business Analytics
